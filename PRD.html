Algorithm

급매 판별 방식

중개사에게 판단을 맡긴다 - 키워드 기반 감지

핵심 철학

급매 여부는 중개사가 판단한다

  • 중개사가 해당 단지 시세를 가장 잘 알고 있음
  • 이미 "급매" 표기하는 관행이 있음
  • 알고리즘 복잡도 최소화 → MVP 출시 속도 향상

감지 로직

IF "급매" OR "급처분" OR "급전"
  IN (atclFetrDesc, tagList)
→ 급매 매물로 분류
→ 사용자에게 알림 발송

감지 키워드

키워드우선순위설명
급매1순위가장 일반적
급처분1순위급하게 처분
급전2순위급전 필요
급히2순위급하게 매도
마이너스피3순위P2 검토

피드 내 정렬 기준

  • 1순위: 신선도 (최신 매물 우선)
  • 2순위: 같은 날짜면 호가 낮은 순
