## 4. BM 아이디어

### Tier 1: 바로 가능
| BM | 수익 모델 |
|----|----------|
| 전면 광고 | 매물 상세→목록 복귀 시 (CPM 5K~12K) |
| 보상형 광고 | "실거래 비교 보기" 전 광고 시청 (CPM 10K~25K) |
| 급매 알림 프리미엄 | 관심단지 무료 1개 / 유료 무제한 (월 2,900~4,900원) |

### Tier 2: 약간의 개발 필요
| BM | 수익 모델 |
|----|----------|
| 프리미엄 분석 | 적정가 산출, 투자 수익률 시뮬레이션 (월 9,900원) |
| 급매 리포트 | 주간 지역별 급매 동향 PDF (건당 1,900원) |
| 중개사 광고 | 단지 상세에 지역 중개사 프리미엄 노출 (CPC) |

### Tier 3: 높은 잠재력
| BM | 수익 모델 |
|----|----------|
| 중개사 B2B | 급매 리스트 + 매도자 데이터 (월 49K~99K) |
| 부동산 데이터 API | 프롭테크 업체에 API 판매 |
| 매물 추천 엔진 | 맞춤 추천 → 계약 성사 시 CPA |

### 수익화 우선순위
```
1순위: 보상형 + 전면 광고 (배너보다 5~15배 수익, 개발 쉬움)
2순위: 급매 알림 프리미엄 (기능성 메시지 API 연동)
3순위: 프리미엄 분석 구독 (DB에 데이터 이미 있음)
```

## 5. 마케팅 전략

### Phase 1: 0 → 1,000 DAU (첫 3개월)
| 채널 | 전략 |
|------|------|

| 바이럴 기능 | 급매 카드 카톡/토스 공유 |

### Phase 2: 1,000 → 10,000 DAU (3~6개월)
| 채널 | 전략 |
|------|------|
| 토스 푸시 A/B 테스트 | CTR 최적화, 세그먼트 세분화 |
| 주간 뉴스레터 | 카카오톡 채널 "이번 주 급매 TOP 10" |
| 콘텐츠 마케팅 | 단지별 급매 분석 블로그 |
| 유튜버 협업 | 급매 데이터 제공 → 영상 언급 |

### Phase 3: 10,000+ DAU (6개월~)
| 채널 | 전략 |
|------|------|
| 리텐션 극대화 | 급매 알림 푸시, "오늘의 급매" |
| 수익화 본격화 | 보상형/전면 광고 + 프리미엄 구독 |

### 가장 효과적인 1가지
**토스 스마트발송 광고성 메시지** — 3,000만 사용자에 직접 푸시, 세그먼트 타겟팅, 월 10만건 무료, 토스 내 서비스라 전환 허들 낮음

## 6. 시장 판단

### 긍정적
- 금리 높고 거래량 줄어 급매 실제로 많은 시기 — 수요 높음
- "급매 전문" 서비스는 전문 플레이어 부재
- "관심단지 급매 푸시"는 네이버가 안 해주는 기능

### 부정적
- 네이버/아실/호갱노노 거인들과 경쟁
- 공공데이터 기반이라 데이터 해자(moat) 없음
- 토스 미니앱은 습관적 접속 어려움

### 결론
급매 앱 자체의 천장은 낮지만, 데이터/기술/경험의 가치는 높음.
진지하게 수익화하려면:
1. 별도 웹으로 SEO 유입 확보
2. 커뮤니티로 리텐션 구축
3. B2B(중개사 툴) 또는 데이터 API로 피벗 검토

## 7. 별도 웹/앱 + 토스 통합 전략

### 토스 정책
- 미니앱에서 자사 앱 설치 유도 **금지**
- 미니앱에서 외부 웹 링크 이동 **제한**
- 반대 방향(별도 웹 → 토스 언급)은 **가능**

### 구조
```
별도 웹 (estate-radar.kr)          토스 미니앱
  ┌─────────────────┐            ┌──────────────────┐
  │ 게시판 (메인)    │            │ 급매 탐색 (메인)  │
  │ 급매 탐색        │            │ 게시판            │
  │ SEO 유입         │            │ 토스 푸시 알림    │
  └────────┬────────┘            └────────┬─────────┘
           └──────── 같은 API/DB ─────────┘
```
- DB/API 공유 (Vercel API)
- 별도 웹: 카카오/구글 로그인, SEO 검색 유입
- 토스: 토스 로그인, 스마트발송 푸시

## 8. 게시판 강화 전략

### 데이터 기반 게시판 (차별점)
| 기능 | 설명 |
|------|------|
| 매물 첨부 | 글 쓸 때 급매 매물 카드 첨부 → 토론 유도 |
| 자동 시세 차트 | 단지명 태그 → 자동 시세 그래프 삽입 |
| 급매 지수 위젯 | 지역별 급매 비율 실시간 표시 |
| 예측 투표 | "6개월 후 서울 아파트 가격은?" 상승/보합/하락 |
| 가격 알림 공유 | "이 단지 알림 같이 받을래?" 바이럴 |

### 게시판 카테고리
```
🔥 급매 제보      ← 사용자 제보 "이거 급매 맞나요?"
📉 하락 분석      ← 하락론자 전용, 데이터 기반 토론
📊 시세 토론      ← "00단지 지금 사도 될까?"
🏗️ 분양/입주      ← 신규 분양, 입주 물량 정보
💡 투자 인사이트  ← 실거래/급매 데이터 기반 분석글
🆓 자유 게시판
```

### 자동 콘텐츠 생성 (봇)
| 게시글 | 주기 | 내용 |
|--------|------|------|
| "이번 주 급매 TOP 10" | 매주 월 | 급매 점수 상위 10개 매물 |
| "지역별 급매 동향" | 매주 수 | 급매 비율 변동 분석 |
| "가격 인하 속보" | 매일 | 1억+ 인하 매물 리스트 |
| "실거래 vs 호가 괴리 TOP" | 매주 금 | 가장 비싼/싼 매물 |
