# 실거래 탭 다음 작업 (2026-03-29)

> 이전 세션에서 실거래 탭 기본 구현 완료. 아래 3가지 개선 작업 남음.

---

## 1. 아실 스타일 타임라인 드래그 슬라이더

### 현재
- 5단계 칩 버튼 (1주/2주/1개월/6주/2개월)
- KB 데이터 10주치만 저장

### 목표
아실 가격분석 탭과 동일한 연속 타임라인 드래그:

```
22년 ──────────────────────●── 26년
           1주일 변동    1년 변동
```

- KB API에서 2년치(101주) 전체 데이터 활용
- 슬라이더 좌측: `22년`, 우측: `26년` (최신)
- 드래그하면 "N주 변동" / "N개월 변동" 텍스트 실시간 갱신
- 지도 색상도 드래그에 따라 실시간 변화

### 필요 작업
- KB 수집 스크립트 수정: 10주 → 전체 101주 히스토리 저장
- `kb-price-index-collect.mjs` 재실행
- 프론트 슬라이더 UI 변경 (RtMapExplorer.tsx)
- API `weeks` 파라미터 범위 확대 (최대 100)

### 참고: 아실 UI 구조 (크롤링 결과)
```
- 지도 위: "26년03월16일 ~ 26년03월23일" 날짜 표시
- 드래그 바: 좌측 "1주일 변동", 우측 "1년 변동"
- 아래 타임라인: "22년 ─── 24년 ─── 25년" 연도 눈금
- 범례: 2%상승, 1%상승, 0.1%상승, 변동없음, 0.1%하락, 1%하락, 2%하락
```

## 2. 단지 레벨 C안 구현

### 현재
- 마커: 단지 전체 평당가 중위값 변동률 표시
- 팝업: 평균가, 변동률, 거래 건수만 표시

### 목표 (C안)
- **마커**: 단지 전체 평당가 중위값 변동률 (현재 유지)
- **팝업 클릭 시**: 면적별로 최신 거래 vs 직전 동일면적 거래 비교 상세 표시

### 팝업 UI 예시
```
래미안대치팰리스
서초구 · 2,990세대 · 평균 40.5억

84㎡ (34평)
  3/19 거래: 40억 5,000만 (14층)
  직전 3/16:  41억 9,000만 (4층)
  ▼ 1억 4,000만 (-3.3%)

59㎡ (24평)
  3/11 거래: 28억 (9층)
  직전 2/15:  27억 5,000만 (12층)
  ▲ 5,000만 (+1.8%)

[단지 상세 보기]
```

### 필요 작업
- 새 API: `GET /real-transactions/complex-detail?complexId=123&period=1m`
  - 면적대별 최신 거래 + 직전 동일면적 거래 반환
- RtKakaoComplexMap.tsx 팝업 UI 확장
- 면적대별 카드 컴포넌트

## 3. 상승률 TOP10 캐러셀 (이번주 실거래)

### 현재
- 세로 카드 리스트 (최신순, 전체 거래)
- 스크롤 형태

### 목표
급매 탭의 "오늘 추천 급매"와 동일한 수평 캐러셀:

```
이번주 상승률 TOP10
[서울] [경기] [인천] ...

┌──────────┐ ┌──────────┐ ┌──────────┐
│ 1        │ │ 2        │ │ 3        │
│ 래미안   │ │ XX아파트 │ │ YY아파트 │
│ 84㎡     │ │ 59㎡     │ │ 74㎡     │
│ +5,000만 │ │ +3,000만 │ │ +2,500만 │
│ (+1.8%)  │ │ (+1.2%)  │ │ (+0.9%)  │
└──────────┘ └──────────┘ └──────────┘
  ● ● ○ ○ ○ ○ ○ ○ ○ ○    ◄ ►
```

### 로직
- 동일 단지+동일 면적(±2㎡) 직전 거래 대비 상승률 계산
- 상승률 상위 10건 (하락 제외)
- 시도별 필터

### 필요 작업
- 새 API: `GET /real-transactions/weekly-top?sido=서울시&sort=change_rate_desc&limit=10`
- RtPageClient.tsx에서 기존 세로 리스트 → 캐러셀로 교체
- HomePageClient.tsx의 캐러셀 패턴 재활용 (scroll-snap, 280px 카드, 도트 인디케이터)

## 구현 순서 (권장)

```
1. KB 히스토리 확장 (수집 스크립트 수정 + 재실행)
2. 타임라인 드래그 슬라이더 (프론트)
3. 상승률 TOP10 API + 캐러셀
4. 단지 상세 팝업 API + UI
```
